DELASOF2026 PHP - QUICK START GUIDE

PROJECT OVERVIEW
A complete PHP port of Delasof2026 affiliate shop with product management, admin panel, URL cloaking, and wishlist functionality.

QUICK START (5 MINUTES)

1. EXTRACT PROJECT
   tar -xzf delasof2026_php.tar.gz
   cd delasof2026_php

2. SET PERMISSIONS
   chmod 755 storage
   chmod 755 storage/csv
   chmod 755 storage/logs

3. START LOCAL SERVER
   cd public
   php -S localhost:8000

4. ACCESS APPLICATION
   Frontend: http://localhost:8000/
   Admin:    http://localhost:8000/admin

5. LOGIN TO ADMIN
   Username: admin
   Password: sofaraway

6. UPLOAD CSV FILE
   Admin Panel → CSV Tab → Upload your product CSV file

MAIN PAGES

/sitemap.xml ............... XML sitemap

API ENDPOINTS
GET  /api/config ..................... Get configuration
POST /api/config ..................... Save configuration (admin)
POST /api/auth/login ................. Admin login
GET  /api/auth/logout ................ Admin logout
GET  /api/products ................... List products
GET  /api/products?action=search ..... Search products
GET  /api/products?action=category ... Get category products
GET  /api/csv ........................ List CSV files
POST /api/csv ........................ Upload CSV (admin)
DELETE /api/csv ...................... Delete CSV (admin)
